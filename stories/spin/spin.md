## **spin** 加载中
```
 import { Spin } from 'dh-c'
```
| props | 说明 | 参数类型 | 默认值|
|-------|-----|---------|------|
| size  | 组件大小，可选值为 small default large | string | 'default' |
| spinning | 是否旋转 | boolean | true |
| tip | 当作为包裹元素时，可以自定义描述文案 | string | - |
| dely | 延迟显示加载效果的时间（防止闪烁） | number(毫秒) | - |
| wrapperClassName | 包装器的类属性 | string | - |
